## About Argon PC

Argon PC is a Windows-focused Roblox utility for loading and executing Lua scripts through a straightforward desktop interface. Built for Windows 10 and Windows 11, it combines rapid script loading, one-click injection, and an included script hub for keeping frequently used workflows together.

Rather than presenting a large, resource-heavy suite, Argon PC aims to provide a compact tool with the essential executor features in one package. Its functionality includes batch execution, locally saved queues, automatic updates, and an integrated debugger, giving users a focused environment for script-related tasks.
